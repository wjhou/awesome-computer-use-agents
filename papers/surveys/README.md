# Survey Papers on Computer Use Agents

Comprehensive survey and review papers covering the field of GUI agents and computer use.

## Major Surveys

### Large Language Model-Brained GUI Agents: A Survey
- **Authors**: Various
- **Date**: November 2024 (Updated May 2025)
- **Link**: [arXiv:2411.18279](https://arxiv.org/abs/2411.18279)
- **Tags**: `survey` `llm` `gui` `comprehensive`

This comprehensive survey explores the historical evolution, core components, and advanced techniques of LLM-brained GUI agents. These agents can interpret complex GUI elements and autonomously execute actions based on natural language instructions.

**Key Topics Covered**:
- Historical evolution from rule-based to LLM-based approaches
- Core components: perception, planning, action
- Training methods and data collection
- Evaluation benchmarks
- Applications across web, mobile, desktop

---

### GUI Agents: A Survey
- **Authors**: Various
- **Venue**: ACL 2025 Findings
- **Link**: [arXiv:2412.13501](https://arxiv.org/abs/2412.13501)

**Key Contributions**:
- Taxonomy of GUI agent architectures
- Comparison of evaluation benchmarks
- Analysis of training paradigms

### JARVIS or Ultron? Safety and Security Threats of Computer-Using Agents
- **Authors**: Various
- **Date**: May 2025
- **Link**: [arXiv:2505.10924](https://arxiv.org/abs/2505.10924)
- **Tags**: `survey` `safety` `security` `threats`

### How Smart Is Your GUI Agent? A Framework for the Future of Software Interaction
- **Authors**: Various
- **Date**: February 2026
- **Link**: [arXiv:2602.11514](https://arxiv.org/abs/2602.11514)
- **Tags**: `survey` `autonomy` `framework` `evaluation`

Conceptual framework that introduces GUI Agent Autonomy Levels (GAL) for computer-use systems.

**Key Contributions**:
- Defines six GUI Agent Autonomy Levels to separate capability, responsibility, and risk.
- Makes autonomy explicit across web, desktop, and mobile software interaction.
- Reframes benchmark design and deployment trust as autonomy-level questions, not only model-quality questions.
